# 🗒️ Notes App

A small CRUD app for creating, editing, and deleting notes.

🔗 Demo: https://katart2001.github.io/Notes/ 

## Features
- Add a note (title + text)
- Edit and delete notes
- Data storage via MockAPI
- Responsive design

## Tech stack
- HTML, CSS/SCSS
- JavaScript (fetch API, CRUD)
- MockAPI
- GitHub Pages
